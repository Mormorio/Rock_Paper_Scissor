Contents in the folder:
1)RPS_Dataset.ipynb: code for dataset collection and preprocessing for gesture recognition.
2)RPS_Model_Training.ipynb: code for construction and training of the gesture recognition model
3)model.h5: the gesture recognition model (inside the folder models)
4)Game.ipynb: Main code for face detection, user identification and management of the rock-paper-scissors game.
5)ICT_Project.pdf: slides for the project presentation.
